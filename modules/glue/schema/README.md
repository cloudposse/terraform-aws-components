# Component: `glue/schema`

This component provisions Glue schemas.

## Usage

**Stack Level**: Regional

```yaml
components:
  terraform:
    glue/schema/example:
      metadata:
        component: glue/schema
      vars:
        enabled: true
        name: example
        schema_name: example
        schema_description: "Glue schema example"
        data_format: JSON
        glue_registry_component_name: "glue/registry/example"
```
