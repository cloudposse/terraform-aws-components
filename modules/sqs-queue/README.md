# Component: `sqs-queue`

This component is responsible for creating an SQS queue.

## Usage

**Stack Level**: Regional

Here's an example snippet for how to use this component.

```yaml
components:
  terraform:
    sqs-queue:
      settings:
        spacelift:
          workspace_enabled: true
      vars:
        enabled: true
```
