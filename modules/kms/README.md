# Component: `kms`

This component is responsible for provisioning a KMS Key.

## Usage

**Stack Level**: Regional

Here's an example snippet for how to use this component.

```yaml
components:
  terraform:
    kms:
      settings:
        spacelift:
          workspace_enabled: true
      vars:
        enabled: true
```
