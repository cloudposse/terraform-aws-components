# Component: `strongdm`

This component provisions [strongDM](https://www.strongdm.com/) gateway, relay and roles 

## Usage

**Stack Level**: Regional

Use this in the catalog or use these variables to overwrite the catalog values.

```yaml
components:
  terraform:
    strong-dm:
      vars:
        enabled: true
```
